Topological Dependency Grammar

Parser And Grammar Development Environment
Version 1.2

Denys Duchier and Ralph Debusmann

This package implements a parser and grammar development environment for Topological Dependency Grammar (TDG), a grammar formalism first described in e.g. (Duchier and Debusmann 2001) and more fully in (Debusmann 2001). TDG grammars can be written in a declarative grammar specification language. This package also includes a graphical user interface to facilitate grammar development.

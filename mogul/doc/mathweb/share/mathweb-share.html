Purpose

These functors provide basic functionality used by other Mathweb packages.

Installation

This package can be installed by following the usual configure, build, install procedure, i.e. by executing a the shell: ./configure make install By default, all files of the package are installed in the user's ~/.oz directory tree. In particular, all modules are installed in the user's private cache.

Example

killtree.exe <pid>
kills the process <pid> and all its children.

benchmark.exe <file>
compiles the Oz source in <file> which must result in a procedure/1, executes it 10 * 10000 times and prints some time statistics.

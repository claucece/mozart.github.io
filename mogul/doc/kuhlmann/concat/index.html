ConCat: A Constraint Workbench for Categorial Grammar

ConCat is planned as a workbench for the application of constraint programming techniques to the processing of categorial grammars. In the current release, it includes components for a parser for resource-sensitive categorial grammar.

Detailed information about the theoretical foundation that ConCat is built on can be found in Marco Kuhlmann's MSc thesis, Towards a Constraint Parser for Categorial Grammar.

To explore the system, it is recommended that you study the source code (which is fairly well commented) and run the file TEST.oz in the examples subdirectory of the distribution. This will give you an Explorer window, which has been amended by various information actions.

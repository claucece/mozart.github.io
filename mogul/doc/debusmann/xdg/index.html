XDG parser system

The XDG parser system is available in source form (for all operating systems) or in binary form (currently only for Windows). The manual of the XDG parser system manual is available in various formats.

Source

In source form, you can download the XDG parser system as an ozmake package: debusmann-xdg.pkg.

Binary

Windows users can download the XDG parser system in binary form as a zip archive: debusmann-xdg.zip. This version is compiled for Mozart 1.3.1.

Manual

You can download the manual of the XDG parser system either as:

  • a tgz archive containing the manual in HTML format: debusmann-xdg-manual_html.tgz
  • a tgz archive containing the manual in info format: debusmann-xdg-manual_info.tgz
  • in PDF format: debusmann-xdg-manual.pdf
  • in gzipped Postscript format: debusmann-xdg-manual.ps.gz

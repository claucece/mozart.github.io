Mozart Installer (alpha)

Andreas Franke

provides
~/.oz/bin/installer.exe
requires
http://www.info.ucl.ac.be/people/ned/qtk/QTk.ozf

Purpose

With this program you can

  • install Oz-functors from the web into your (local or global) Oz cache
  • create applets that do this installation

Usage

installer.exe --title <PackageTitle>
      --resolve <ResolverSpec>
      <FunctorUrl>, ...
      [ -o <Package>.oza]

Installation

This package can be installed by following the usual configure, build, install procedure, i.e. by executing a the shell: ./configure make install By default, all files of the package are installed in the user's ~/.oz directory tree. In particular, all modules are installed in the user's private cache.

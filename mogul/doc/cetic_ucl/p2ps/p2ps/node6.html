6 Announcements

27 Oct. 2004 Set up the P2PS mailing list

On this mailing list developers and users of the P2PS library can exchange messages about their experience with the library (e.g., achievements or bugs found) as well as possible suggestions for its improvement.

29 June 2004 Introduce reliable communication

Introduce the first reliable communciation primitive into P2PS. Take a look to the method rsend of class P2PS.p2pServices.

30 Oct. 2003 The release of P2PS version 1.0 and adding it to MOGUL

A new release of the P2PS (version 1.0). The main characteristic of this release is the improvement of the API which now, we think, looks more natural to the developper. Although independent of the underlying peer-to-peer technology, P2PS currently only supports Tango[1] protocol (the former called GS-Chord[3]).

8 May 2003 The first release of P2PS

The first release of P2PS (beta version) as a Mozart library that offers different primitives and services for peer-to-peer applications. It implements both DKS[5] and GS-Chord[3] protocols (Chord[6] based protocols). By employing the overlay network primitives, it provides point-to-point, multicast and broadcast communcation primitives.

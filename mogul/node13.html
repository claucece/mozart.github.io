3.3 Developing and packaging contributions

The job of developing and packaging a contribution so that it is easy to install anywhere is ``in principle'' not that straightforward. Fortunately, we have already done most of the work for you. We provide utilities that make this task very simple: ozskel for the old obsolete way, ozmake for the new recommended way.

This package implements a parser system and graphical user interface for Extensible Dependency Grammar (XDG). It is the successor of the TDG parser and grammar development environment.

The implemented grammar formalism is described in A Meta-Grammatical Framework for Dependency Grammar, and in Dependency Grammar as Graph Description. A description of the parser system can be found in A Parser System for Extensible Dependency Grammar.

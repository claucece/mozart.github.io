Help

blurb:A hypertext widget with parsing facilities and support for creating help windows.
author:Donatien Grolaux
contact:Donatien Grolaux
documentation:index.html
download:grolaux-help-1.0.tar.Z
provides:http://www.info.ucl.ac.be/people/ned/help/Help.ozf
The Help module provides an easy to use and highly configurable hypertext system with a dedicated support to create graphical help windows. This module is highly configurable, from the way the text is parsed to the actions to be execute depending on the tags. This module defines a default configuration so that user can use it right away.

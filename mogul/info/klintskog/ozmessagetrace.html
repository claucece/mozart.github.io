Ozmessagetrace

type:package
id:mogul:/klintskog/ozmessagetrace
section:mogul:/klintskog
blurb:A tool for viewing logs from distributed Mozart
author:Erik Klintskog
category:tool
download:klintskog-ozmessagetrace.tgz
provides:[tool] ~/.oz/bin/ozmessagetrace

The Message tracer helps the user to understand the logs produced by Mozart sites doing distributed communication. The Message tracer takes a list of files as argument and sorts the send and receive events found in the files in a casual order. The result is displayed in a GUI.

ozmessagetrace file1 ... fileN

The loging can be switched on and off by the properties

Since mozart-1.3.0: Inspector with alternative defaults.

For mozart-1.2.x: This package provides a fully functional Inspector for Mozart-1.2.x. This inspector it taken from Mozart-1.3.x and supports:

  1. inspecting in deep spaces
  2. displays strings nicely (by default as everybody would expect)
  3. and supports the creation of new inspector windows with many user definable options.

The default inspector of the stable Mozart-1.2.x branch is not missing these features in contrast to the devellopment branch Mozart-1.3.x.
